== Quick Start

include::attributes.adoc[]

To start, you need to add the dependency into your maven `pom.xml` or gradle `build.gradle`.

[source,xml,role="primary"]
[subs=attributes+]
.pom.xml
----
<dependency>
    <groupId>com.yaml-validator</groupId>
    <artifactId>yaml-validator-all</artifactId>
    <version>{project-version}</version>
</dependency>
----

[source,groovy,role="secondary"]
[subs=attributes+]
.build.gradle
----
implementation("io.github.kezhenxu94:yaml-validator-all:{project-version}")
----
